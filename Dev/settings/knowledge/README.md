# Knowledge Base - AI Trader

> Ovaj folder sadrži domensko znanje koje AI koristi.
> Dodaj svoje bilješke, lekcije, i pravila ovdje.

## Struktura

| Fajl | Sadržaj |
|------|---------|
| `forex_basics.md` | Osnove forex tradinga |
| `risk_rules.md` | Tvoja osobna risk pravila |
| `lessons.md` | Naučene lekcije iz prošlih tradeova |
| `pairs_notes.md` | Bilješke o specifičnim parovima |

## Kako koristiti

1. Editiraj postojeće fajlove
2. Dodaj nove `.md` fajlove
3. AI će ih automatski učitati

## Tip

Zapisuj lekcije nakon svakog lošeg tradea:
- Što je pošlo po zlu?
- Što si mogao drugačije?
- Pravilo za budućnost
